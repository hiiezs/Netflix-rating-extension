Solving Netflix's lack of ratings from IMDB and Rotten Tomatoes with a browser extension that injects ratings directly into the UI using the OMDB API.

What it does: Inject IMDB and Rotten Tomatoes ratings to your Netflix, so you know if that movie or show is worth watching or not.

HOW?
1. Install the "TamperMonkey" extension to your browser
2. Choose Add script and input the script.js file content from this repo
3. Turn the script on whenever you need to use it

Note: Make sure you're in Netflix (it doesn't work for other streaming sites yet), and re-load when it isn't working.
